
-   ⚛️ Tech Stack: React.js, Node.js, Express.js, MongoDB
-   🔥 Build an API
-   📱 Responsive UI With React.js and Tailwind CSS
-   🐞 Error Handling
-   🌐 Deployment
-   🚀 And Many More Cool Features

### Setup .env file

```shell
MONGO_URI=your_mongo_uri
PORT=8000
```

### Run this app locally

```shell
npm run build
```

### Start the app

```shell
npm run start
```
